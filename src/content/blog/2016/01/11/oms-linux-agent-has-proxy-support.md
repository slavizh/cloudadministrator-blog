I was browsing trough GitHub and I’ve noticed that OMS Linux Agent documentation has been updated. In the change log we can see the following changes:

-   Agent HTTP(s) proxy support
-   Improved performance and message throughput.
-   Disk-buffered output
-   Ubuntu 15.10 support

    According to the documentation OMS Linux agent is now 1.1.0-2.

    It seems documentation has been significantly expanded as well.

    You can find the GitHub documentation [here](https://github.com/Microsoft/OMS-Agent-for-Linux).
